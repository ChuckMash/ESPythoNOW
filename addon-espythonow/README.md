[ESPythoNOW - ESP-NOW to MQTT Bridge](https://github.com/ChuckMash/ESPythoNOW)


Communicate *directly* with ESP-NOW capable devices via supported Internal/External Wireless Interfaces.

* Send and receive ESP-NOW messages directly to and from Home Assistant / MQTT and ESP8266 / ESP32 / etc devices
* Supports wide variety of Wireless Interfaces
* Monitor all ESP-NOW messages
* Supports ESP-NOW v1.0 and v2.0
* Supports sending and receiving encrypted ESP-NOW messages!
